# 数据源参考

本文件描述了`quant-investor`技能使用的数据源。

## 主要数据源

- **Tushare**: 作为本技能获取A股和美股数据的核心数据源。它提供了丰富的财经数据，包括股价、财务报表、估值指标等。

  - **API Token**: 使用Tushare前，必须在`config.ini`中配置有效的API token。
  - **数据权限**: Tushare的数据权限基于积分，部分高阶数据可能需要较高积分。请确保您的账户有足够权限。

## 备用免费数据源

在Tushare数据不可用或不完整时，本技能将尝试使用以下免费数据源作为补充：

- **yfinance**: 一个流行的Yahoo Finance API的Python封装，可用于获取美股和部分全球市场数据。
- **AKShare**: 一个开源财经数据接口库，提供A股、港股、美股等市场的实时和历史数据。

## 已排除数据源

- **Wind**: 由于用户反馈接口不稳定，本技能已明确排除使用Wind作为数据源。
