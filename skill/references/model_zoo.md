# 模型库参考

本文件介绍了`quant-investor`技能支持的AI模型。用户可以在`config.ini`中选择和配置希望使用的模型。

## 1. 传统机器学习模型

这类模型适用于结构化数据，训练速度快，可解释性较好。

- **XGBoost**: 一种高效的梯度提升决策树模型，广泛用于各类预测任务。
- **LightGBM**: 另一种高性能的梯度提升框架，特别是在大规模数据集上表现出色。
- **随机森林 (Random Forest)**: 一种集成学习模型，通过构建多个决策树并取其平均结果来提高预测准确性和控制过拟合。

## 2. 深度学习模型

深度学习模型擅长处理时序数据和捕捉非线性关系。

- **长短期记忆网络 (LSTM)**: 一种特殊的循环神经网络（RNN），非常适合用于处理和预测时间序列数据，如股票价格。
- **Transformer**: 最初用于自然语言处理的模型，其自注意力机制也被证明在金融时间序列分析中非常有效。

## 3. 强化学习模型

强化学习模型通过与环境的交互来学习最优的交易策略。

- **深度Q网络 (DQN)**: 将深度学习与Q学习相结合，用于在复杂的、高维度的状态空间中学习策略。
- **近端策略优化 (PPO)**: 一种先进的策略梯度方法，通过限制每次更新的策略变化幅度来提高训练的稳定性。

在`model_training.py`脚本中，将根据配置文件中的选择来加载和训练相应的模型。
