# 特征库参考

本文件定义了`quant-investor`技能在进行模型训练时可以使用的特征。这些特征分为三类：基本面特征、技术面特征和另类数据特征。

## 1. 基本面特征

基本面特征反映了公司的财务状况和内在价值。

- **估值指标**: 市盈率 (PE)、市净率 (PB)、市销率 (PS)、股息率等。
- **盈利能力**: 净资产收益率 (ROE)、总资产报酬率 (ROA)、毛利率、净利率等。
- **成长能力**: 营收增长率、净利润增长率等。
- **财务质量**: 资产负债率、流动比率、速动比率等。

## 2. 技术面特征

技术面特征基于历史价格和成交量数据，用于捕捉市场情绪和趋势。

- **动量指标**: 移动平均线 (MA)、指数移动平均线 (EMA)、MACD、RSI、KDJ等。
- **波动率指标**: 布林带 (Bollinger Bands)、平均真实波幅 (ATR)等。
- **成交量指标**: 成交量移动平均 (VMA)、能量潮 (OBV)等。

## 3. 另类数据特征

另类数据提供了传统财务数据之外的独特视角。

- **市场情绪**: 基于新闻头条、社交媒体评论的情绪分析得分。
- **宏观经济**: GDP增长率、通货膨胀率 (CPI)、利率等宏观经济指标。

在`feature_engineering.py`脚本中，您可以选择启用或禁用这些特征的计算。
