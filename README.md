# MyQuant - AI量化投资工具箱

一个端到端的AI量化投资框架，融合了统计学、贝叶斯推断、现代投资组合理论、投资大师智慧和大型语言模型增强分析。

**当前版本**: V3.1 (动态智能框架)
**核心理念**: 全球化一手数据驱动 + LLM深度思辨 + 动态智能决策 + 全方位风险管理

---

## 🎯 项目特色

- ✅ **全球化数据 (V3.0)**：同时支持A股和美股市场，覆盖宏观、行业、个股、**期货期权**全维度数据
- ✅ **一手数据驱动**：直接从FRED、Tushare、Finnhub、yfinance等专业数据源获取原始数据
- ✅ **动态智能决策 (V3.1)**：内置**因子衰减监控**和**高级组合优化**模块，实现从“静态分析”到“动态决策”的闭环。
- ✅ **多Agent深度思辨**：内置模拟投研团队的多Agent辩论系统，从财务、行业、护城河、估值、风险等多个维度对公司进行深度、结构化的基本面分析。
- ✅ **全方位风险管理**：内置专业风险管理模块，提供超过20种风险指标计算、因子风险分解、情景分析和压力测试。
- ✅ **持久化存储**：内置持久化数据存储系统，一次下载，永久使用，极大提升数据获取效率。
- ✅ **科学严谨**：基于统计显著性检验，避免过拟合和数据挖掘偏差
- ✅ **理论完备**：整合现代投资组合理论(MPT)、贝叶斯推断、因果推断
- ✅ **工业级架构**：借鉴微软Qlib框架设计，表达式引擎、特征缓存、增强回测

---

## 🆕 V3.1 新特性：动态智能框架

V3.1版本引入了全新的**动态智能框架**，解决了量化策略在现实世界中面临的两大核心痛点：**因子有效性衰减**和**静态权重无法适应市场变化**。

### 核心特性

| 特性 | 描述 |
|:---|:---|
| **因子衰减监控** | 实时监控策略在各风险因子上的暴露度，通过计算**因子半衰期**，自动预警因子失效风险，并提示进行策略再平衡。 |
| **高级组合优化** | 内置多种现代组合优化算法（如**均值-方差、风险平价、Black-Litterman**），能够根据最新的市场预期和风险状况，动态计算最优投资组合权重。 |

### 2. 运行示例 (V3.1)

使用新的`PortfolioOptimizer`对资产组合进行优化。

```python
import sys
import pandas as pd
import numpy as np

sys.path.append("scripts/v3.1")
from intelligence_layer import PortfolioOptimizer

# 1. 准备预期收益和协方差矩阵
assets = ["股票A", "股票B", "股票C"]
expected_returns = pd.Series([0.12, 0.10, 0.08], index=assets)
vols = np.array([0.20, 0.18, 0.15])
corr = np.array([[1.0, 0.3, 0.2], [0.3, 1.0, 0.5], [0.2, 0.5, 1.0]])
cov_matrix = pd.DataFrame(np.outer(vols, vols) * corr, index=assets, columns=assets)

# 2. 初始化优化器并比较不同方法
optimizer = PortfolioOptimizer()
comparison = optimizer.compare_methods(expected_returns, cov_matrix)

print("不同优化方法对比:")
print(comparison)
```
